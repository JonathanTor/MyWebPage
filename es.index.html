Proyectos Recientes

Dispositivo de Seguridad

Un dispositvo que detecta movimiento y notifica por WhatsApp

Este dispositovo usa un microprosesador como basa. Fue programado usando Python, se uso un sensor de movimiento para la detección y una vez que este era activado se envia una foto por WhatsApp de lo que detecto y se activa una alarma.

Fototrampeo para el bosque

Detectar movimiento e identificar la especie

El departamento de biologia necesitaba que se tomen fotos y luego que estas sean clasificadas segun la especie. Para el hardward se uso un microprocesardor, un modulo de camara y un sensor de movimiento PIR, para el software se uso Python, Open CV y para la clasificacion por especie se uso un herramienta de Open CV(Cascade Classifier), las fotos se subian a Dropbox para que puedan ser analizadas por Internet.

Juego

Puedes controlar a Pacman con tus movimientos

Los niños con problemas de movilidad deben realizar cietos ejecrcicios y deben relizar movimiento como tratamiento para su problema de movilidad. Para ello se uso como base el conocido juego Pacman y con una pelota del color selecionado se puede controlar el juego: Para el desarrollo se uso una computadora y para programarlo se uso Python, Open CV y Pygame
